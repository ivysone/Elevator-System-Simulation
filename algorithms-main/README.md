# Lift Algorithm Coursework

## Overview
This project simulates a lift system using different scheduling algorithms:

- **SCAN**
- **LOOK**  
- **Mylift**  

These algorithms use the data structures defined in `datastructures.py`, and `customer.txt`  
These files are required to run the algorithms.

## How to Run
1. Execute the algorithm files `Scan_main.py` `Look_main.py`
2. A pop-up window will open
3. Start the simulation and adjust the simulation speed
4. Or you can run the simulation floor by floor by clicking on **step**

## Event Log
- **Lift Simulation:**  
  - The elevator always starts at the **1st floor**.  
  - It moves according to the scheduling algorithm.
  -   
- **Final Report:**  
  - Displays the total number of elevator movements.  
  - Shows the status of each customer:  
    - **DELIVERED** → The customer successfully reached their destination.  
    - **WAITING** → The customer has not been served.
